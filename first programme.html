Dokan provides a nice and clean admin overview page where admins can view a summary of their marketplace at a glance. Not only is it visually appealing but it also shows important stats for the admins to get quick insights and take immediate actions. The admin dashboard shows net sales, number of products created,and number of signups, pending vendors,

The dashboard also provides a line chart to compare total sales, number of orders and commission for each day of the month.

The Dokan overview also displays latest blogs and a subscription widget so users can subscribe to the newsletter straight from admin dashboard.
